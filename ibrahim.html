Computer Science contributions to society


There are a number of contributions computer science has offered to the world which resulted in positive innovation. One example of this is with the environment. It first started with bottle deposit schemes. These schemes offered those who bought bottled drinks which were plastic to return them for recycling purposes. This way, those who return their bottles only pay for the beverage and not the plastic bottle itself. These schemes are popular in mainland Europe, but very abundant in Germany.
